## 평가 기준 (5축, 각 10점)

| 축 | 설명 | 경쟁사 기준 |
|----|------|------------|
| 감각묘사 | 시각/청각/촉각 등 감각 디테일 | "냉각수 새어나옴", "LED 고글 번쩍" |
| 행동묘사 | 캐릭터의 물리적 동작 | "벽에서 몸을 떼고 성큼성큼", "관자놀이 툭" |
| 다중캐릭터 | 여러 캐릭터 자연스러운 등장 | T1부터 ZERO+노바 동시 등장 |
| 세계관활용 | 로어북/설정 자연스러운 녹여내기 | 언더벨리, 오라클, 뉴로링크 |
| 스토리긴장감 | 사건/위기 제시로 몰입 유도 | "단속반이 내려오고 있어" |
